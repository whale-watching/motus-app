## Features
- Support for `cm`, `mm`, `in`, `px`, `pt`, `pc`, `em`, `ex`, `ch`, `%`, `rem`, `vw`, `vh`, `vmin`, `vmax`, `deg` units
- Support for standard 
- Scroll hooks

## Installation
As [npm](https://www.npmjs.com/package/motus) package
```
npm i -S motus
```
Standalone script file
```js
<script src="dist/motus.js"></script>
```
